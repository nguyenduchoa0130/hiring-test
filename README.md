# Usage

```bash
# Step 1
git clone https://github.com/nguyenduchoa0130/hiring-test.git
# Step 2
cd hiring-test
# Step 3
npm i
# Step 4
npm start
```

- Open your browser on http://localhost:4200/

# Completed features

- Add task
- Edit task
- Complete Task
- Filter by status, description, username
